# Diplom_work

Диплом описание


Сайт по рекрутингу (поиску и анализ претендента на вакансию)

Похожие сайты:
НН
Linkedin
SuperJob

Функционал

Есть бешенное желание написать сайт, на котором пользователь сможет

- по фильтру искать необходимую им работу из базы данных
- также программа будет мониторить имеющиеся аналогичные сайты (HH, linkedIn и т.д.) на наличие необходимой вакансии
- добавлять в корзину понравившикся вакансии
- определять геолокацию понравившихся вакансий
- загрузить на сайт ряд тестов для определения подходит ли соискатель на искомую вакансию
- по возможности привязать это к эл. почте или мессенджеру для оперативного уведомления
